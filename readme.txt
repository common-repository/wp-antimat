== Description ==

The plugin monitors uncensored russian words in comments and closes them by [censored]. Based on Anti Mate PHP Class from s1ayer.
Catches as a popular classic word mats, and a number of modern ("Albanian") words.

Плагин отслеживает нецензурные выражения в комментариях и закрывает их надписью [censored]. Базируется на Anti Mate PHP Class от s1ayer (www.spg.arbse.net).
Отлавливает как классические словоформы популярных матов, так и ряд современных ("албанских") словоформ.

== Installation ==

1. Загрузить `wp-antimat.php` в папку `/wp-content/plugins/`
2. Активировать плагин в разделе 'Плагины' анминистарторской панели
